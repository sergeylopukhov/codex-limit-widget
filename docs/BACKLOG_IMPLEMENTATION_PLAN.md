# План реализации идей из BACKLOG.md

Дата: 22 июля 2026

## Принятые решения

- Реализовать все три идеи в текущем цикле.
- Нажатие на виджет открывает основное приложение (окно настроек).
- Уведомления — системные уведомления macOS; по умолчанию выключены.
- При первом включении предлагаются пороги 10% и 15%; новые пороги добавляются кнопкой `+` с шагом +5% от предыдущего, всего до пяти значений.
- Каждый порог срабатывает один раз для соответствующего окна до его следующего сброса.
- «Системное» оформление применяется к окну приложения и виджету: тёмная macOS — DARK, светлая — BEIGE.
- Завершение: стабильный релиз `1.2.0` после проверки beta-версии.

## Этапы

1. Добавить в модель настроек системное оформление и настройки уведомлений с обратно совместимым Codable-декодированием.
2. Добавить менеджер системных уведомлений macOS: запрос разрешения при включении, дедупликация по окну, порогу и времени сброса, обработка недоступного 5-часового окна.
3. Вызывать проверку уведомлений после каждого успешного обновления лимитов и добавить настройки: включатель и до пяти редактируемых порогов.
4. Сделать стили окна динамическими для режима «Системное»; передавать фактический DARK/BEIGE дизайн в меню-бар и настройки.
5. Сделать виджет кликабельным через URL приложения; обработать URL и показать основное окно.
6. Выполнить сборку без подписи и проверку diff. Не менять существующие незакоммиченные правки обновления виджета.
